hexo+github+vercel部署博客

Hexo

Hexo: 一个快速, 简洁且高效的博客框架

因为 Hexo 是基于 Node 的, 所以要想使用 Hexo 就需要按照 Node.js

Node.js下载地址: https://nodejs.org/en

安装好Node.js后 打开cmd命令行窗口输入以下命令:

输入: node -v
输出: 	-> v18.17.0
输入: npm -v
输出: 	-> 9.6.7

如果没有报错, 输出版本号后就说明 Node.js 安装好了, 接下啦, 我们就可以愉快的安装 Hexo 了!

首先, 如果没有科学上网, 我们需要换一下国内源, 不然下载很慢

在命令行中执行:

npm config set registry https://registry.npm.taobao.org

之后, 我们进行安装 Hexo 脚手架

npm install hexo-cli -g

如果出现Hexo的相关信息则说明Hexo安装成功

接下来创建一个存放博客的文件夹, 顺序执行以下命令:

hexo init blog
cd blog
npm install
