第二代hysteria节点搭建教程

2023/9/17 教学 科学上网 vps

前言:

歇斯底里(hysteria)是底层传输基于udp的加密代理协议，对比目前使用的ss、vmess、vless、trojan等协议，歇斯底里在绕过防火墙审查的同时还能大幅提升上网速度，这是因为歇斯底里修改了quic的拥塞控制算法。让我们在配置文件中手动设置发送速率和接收速率，如果链路发生了拥堵，其他quic用户的拥塞控制是bbr或者cubic，会主动降低发送速率，而歇斯底里用的 拥塞控制算法不管网络是否拥堵，始终按照我们在配置中设定的速率收发数据，当别人降速避让的时候，我们全速发送，在尽力而为的交付规则下，这样能抢占更多的带宽，实现网络提速，这就是歇斯底里能大幅提升网速的原因。

服务器相关指令:

#一键安装Hysteria2
bash <(curl -fsSL https://get.hy2.sh/)
	
#生成自签证书
openssl req -x509 -nodes -newkey ec:<(openssl ecparam -name prime256v1) -keyout /etc/hysteria/server.key -out /etc/hysteria/server.crt -subj "/CN=bing.com" -days 36500 && sudo chown hysteria /etc/hysteria/server.key && sudo chown hysteria /etc/hysteria/server.crt

#启动Hysteria2
systemctl start hysteria-server.service
#重启Hysteria2
systemctl restart hysteria-server.service
#查看Hysteria2状态
systemctl status hysteria-server.service
#停止Hysteria2
systemctl stop hysteria-server.service
#设置开机自启
systemctl enable hysteria-server.service
#查看日志
journalctl -u hysteria-server.service

服务器配置文件:

cat << EOF > /etc/hysteria/config.yaml
listen: :443 #监听端口

#使用CA证书
#acme:
#  domains:
#    - a.com #你的域名，需要先解析到服务器ip
#  email: test@sharklasers.com

#使用自签证书
#tls:
#  cert: /etc/hysteria/server.crt
#  key: /etc/hysteria/server.key

auth:
  type: password
  password: 123456 #设置认证密码
  
masquerade:
  type: proxy
  proxy:
    url: https://bing.com #伪装网址
    rewriteHost: true
EOF

客户端配置文件:

server: ip:443
auth: 123456

bandwidth:
  up: 20 mbps
  down: 100 mbps
  
tls:
  sni: a.com
  insecure: false #使用自签时需要改成true

socks5:
  listen: 127.0.0.1:1080
http:
  listen: 127.0.0.1:8080

步骤:

  1. 打开我们的vps，输入一键安装

    bash <(curl -fsSL https://get.hy2.sh/)

  2. 在vps里生成自签证书

    openssl req -x509 -nodes -newkey ec:<(openssl ecparam -name prime256v1) -keyout /etc/hysteria/server.key -out /etc/hysteria/server.crt -subj "/CN=bing.com" -days 36500 && sudo chown hysteria /etc/hysteria/server.key && sudo chown hysteria /etc/hysteria/server.crt

  3. 设置配置文件, 有域名使用CA证书, 没有域名使用自签证书

    有域名的CA证书配置:

    cat << EOF > /etc/hysteria/config.yaml
listen: :443 #监听端口

#使用CA证书
acme:
  domains:
    - coldeyes.pp.ua #你的域名，需要先解析到服务器ip
  email: coldeyesce@gmail.com  #可以填写你自己的邮箱

auth:
  type: password
  password: 123456 #设置认证密码
  
masquerade:
  type: proxy
  proxy:
    url: https://bing.com #伪装网址
    rewriteHost: true
EOF

    没有域名的自签证书配置:

    cat << EOF > /etc/hysteria/config.yaml
listen: :443 #监听端口

#使用自签证书
tls:
  cert: /etc/hysteria/server.crt
  key: /etc/hysteria/server.key

auth:
  type: password
  password: 123456 #设置认证密码
  
masquerade:
  type: proxy
  proxy:
    url: https://bing.com #伪装网址
    rewriteHost: true
EOF

  4. 启动服务

    先查看歇斯底里2启动状态

    systemctl status hysteria-server.service

    没启动的话，启动歇斯底里2

    #启动Hysteria2
    systemctl start hysteria-server.service
    #重启Hysteria2
    systemctl restart hysteria-server.service

    启动好了后，不一定可以使用，因为没有出现 server up and runing的提示, 可以等待一下，重新查看启动状态

    如果重新查看状态发现启动失败，大概率是没有放行端口，歇斯底里2需要放行配置443端口和80端口，或者关闭防火墙后重新启动歇斯底里2

    查看端口

    ufw status

    关闭防火墙

    ufw disable

    如果还是失败，且没有域名，可以使用自签证书，修改配置文件记得重启歇斯底里2

  5. 客户端配置

    下载v2rayn

    下载Hysteria 2

    解压下载好的v2rayn压缩包，找到Hysteria目录将下载好的Hysteria 2执行文件替换掉v2rayn里的

    新建一个文本文件，将客户端配置文件粘贴进去

    server: ip:443 #ip改为你自己的vpsip
auth: 123456 #密码和vps配置的认证密码一致

bandwidth:  #上传和下载速度 不清楚可以关闭代理去测速网站(比如speedtest.cn)测试一下
  up: 32 mbps
  down: 85 mbps
  
tls:
  sni: coldeyes.pp.ua  # vps配置的是CA证书使用配置的域名，使用的自签证书，使用配置的伪装域名，并且下面设置为true
  insecure: false #使用自签时需要改成true

socks5:
  listen: 127.0.0.1:1080
http:
  listen: 127.0.0.1:8080

    打开v2rayn,新建一个自定义服务器，并且配置

    接着就测试看能不能科学上网了
